HOW TO INSTALL (VS Code + GitHub Copilot / Claude extension)
-------------------------------------------------------------
1. Locate your global skills folder:
   Windows : C:\Users\<you>\.claude\skills\
   Mac/Linux: ~/.claude/skills/

2. Copy the skill sub-folders from by-source/ into that directory.
   Example:
     copy by-source\global-claude\dotnet-sast-pipeline  ->  C:\Users\<you>\.claude\skills\

3. Restart VS Code (or reload the Copilot/Claude extension).

4. In the chat, type the skill trigger (see each SKILL.md for details).
   Example: "Run dotnet SAST scan on D:\MyProject\MySolution.sln"
