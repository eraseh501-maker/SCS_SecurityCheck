# AI 工具共用指引

本專案採用雙層知識結構：

- `docs/90_全域AI開發守則.md`
  由外部 `AI-Knowledge` 全域知識庫同步進來，放跨專案共用的經驗。
- `docs/10_開發回顧與防錯守則.md`
  放這個專案自己的踩坑、商業規則與限制。

## 日常維護

- 跨專案都適用的經驗，先改外部全域知識庫，再同步回 repo。
- 只屬於這個專案的經驗，直接改本 repo 的 `docs/10_開發回顧與防錯守則.md`。
- 建議每次新增 lesson 時都標記 `scope: global` 或 `scope: project`，避免寫錯地方。

## 同步命令

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\sync-ai-knowledge.ps1
```
